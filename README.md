# Antrian Online Rumah Sakit

Aplikasi web antrian online untuk rumah sakit menggunakan Next.js dan Supabase.

## Fitur
- **Halaman Utama**: Booking antrian dengan form pendaftaran
- **Dashboard Petugas**: Manajemen antrian untuk staff rumah sakit
- **Realtime Updates**: Update antrian secara real-time menggunakan Supabase Realtime
- **Login Petugas**: Sistem autentikasi sederhana untuk staff

## Tech Stack

- **Frontend**: Next.js 16, React 19, TypeScript
- **Styling**: Tailwind CSS
- **Backend**: Supabase (PostgreSQL + Realtime)
- **State Management**: React Context API
